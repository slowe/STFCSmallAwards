STFC/PPARC Small Awards 1999-2014

The STFC Public Engagement Small Awards Scheme provides funds for small, local or 'pilot' projects promoting STFC science and technology. Anyone can apply, including grant-funded research groups, STFC research facility users, schools, museums, etc. This page provides a breakdown of STFC Small Award grants awarded between 1999 and 2014. The data have been extracted from PDFs released by STFC. The regions have been determined by the location of the grant holder (not always the region where the money is going to be spent). The code and data for this are on GitHub so please contribute additions/issues there.
